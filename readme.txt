== Description ==

<strong>Widget Actualites Relation Client</strong> vous permet d'ajouter sur votre barre de navigation (sidebar) les derni&egrave;res actualit&eacute;s des douze principaux m&eacute;dias professionnels dans le domaine du management, de la relation client et de la qualit&eacute; de service.

Ce plugin s'adresse &agrave; un public professionnel et français.
Il a &eacute;t&eacute; d&eacute;velopp&eacute; par l'&eacute;quipe du site [contacter.net](http://contacter.net).
Il a &eacute;t&eacute; conçu pour un besoin bien sp&eacute;cifique, nous savons pertinament qu'il touche tr&egrave;s peu d'utilisateurs potentiels mais nous avons tout de même d&eacute;cid&eacute; de partager avec vous ce plugin.

Il vous permet d'afficher sous la forme d'un widget les derni&egrave;res actualit&eacute;s du flux rss que vous avez choisi.
Ce module ne dispose pas pour le moment de version en anglais ou dans une autre langue.

Vous pouvez a tout moment changer la source (flux rss) du widget pour afficher d'autres actualit&eacute;s.
Il est &eacute;galement possible d'ajouter plusieurs widgets pour afficher plusieurs sources sur votre sidebar.

Vous pouvez afficher uniquement les titres des derni&egrave;res actualit&eacute;s ou ajouter une courte description.
Pour en savoir plus sur ce plugin et sur ses diff&eacute;rentes fonctionnalit&eacute;s, vous pouvez vous rendre dans la section "FAQ".

Voici les principaux domaines des m&eacute;dias s&eacute;l&eacute;ctionn&eacute;s :

* Le management en entreprise
* Business BtoB et BtoC
* Les services client et les SAV (service apr&egrave;s vente)
* La relation client et la gestion des crises
* L'entreprenariat en France
* Les centres de contacts et d'appels
* La qualit&eacute; pour les professionnels
* Les conseils informatiques et web (IT)
* La strat&eacute;gie de communication pour votre soci&eacute;t&eacute;
* Contact PRO et particuliers
* Les nouvelles technologies
* Le support professionnel


== Installation ==

1. T&eacute;l&eacute;charger le plugin Widget Actualites Relation Client.
2. D&eacute;zipper le contenu du fichier zip &agrave; l'aide d'un logiciel de d&eacute;compression.
3. Puis uploader le contenu du dossier dans le r&eacute;pertoire /wp-content/plugins/ &agrave; l'aide de votre logiciel FTP.
4. Activer le plugin "Widget Actualites Relation Client" via votre inferface administrateur sur WordPress dans l'onglet 'Extensions'.
5. Rendez-vous dans le menu "Apparence" > "Widgets" et d'ajouter le Widget 'Actu relation client' dans votre ou vos sidebar(s).
6. Vous devez choisir votre source, le type d'affichage et activer ou non diff&eacute;rentes options.
7. Cliquer sur le bouton bleu "Enregistrer" pour faire apparaitre le widget publiquement sur votre blog.


== Frequently Asked Questions ==

= Quelle est la langue du plugin ? =
Ce plugin est disponible uniquement en français.
Il ne dispose pas pour le moment de version anglophone ou dans une autre langue.

= A qui est destin&eacute; ce plugin ? =
Aux professionnels dans l'univers de la relation client.
Ce plugin est destin&eacute; &agrave; un public francophone.

= Est-il possible d'ajouter plusieurs widgets ? =
Il est possible d'ajouter plusieurs widgets pour afficher plusieurs sources d'actualit&eacute;s sur votre sidebar.
Cette option est disponible tout en bas de la page "Widget".

= Quels sont les sources disponibles (flux rss) ? =
* Journaldunet (Management)
* Journaldunet (Business)
* LSA-conso (Service clientele)
* Easi-crm (Relation client)
* Dynamique-mag (Entreprenariat)
* En-contact (Centre de contacts)
* Qualiteperformance (Qualite)
* Lemagit (Conseils IT)
* Relationclientmag (Strategie
* Relationclientmag (Contact)
* Relationclientmag (Technologie)
* Relationclientmag (Professionnels)

= Ce plugin va-t-il ajouter de la publicit&eacute; sur mon blog WordPress ? =
Non, ce plugin vous propose uniquement d'afficher les derni&egrave;res actualit&eacute;s de diff&eacute;rents m&eacute;dias dans le domaine de la relation client.
Ce plugin ne piste pas ses utilisateurs, il n'est pas reli&eacute; &agrave; un service ou serveur tiers.

= Comment supprimer ce plugin ? =
Il vous suffit de le d&eacute;sactiver dans la rubrique 'Extensions' de votre panneau d'administration WordPress.
Ce plugin de stocke pas de cookies sur votre serveur ou sur un serveur tiers.
Vous pouvez &eacute;galement supprimer le dossier "Widget Actualites Relation Client" dans le r&eacute;pertoire /wp-content/plugins/.
